# QA Service - REST API на Go + PostgreSQL



## Стек

- Go (net/http)

- Chi router

- GORM

- PostgreSQL

- Goose (миграции)

- Docker + docker-compose



## API



### Questions

- GET /questions/  

- POST /questions/  

- GET /questions/{id}  

- DELETE /questions/{id}



### Answers

- POST /questions/{id}/answers/  

- GET /answers/{id}  

- DELETE /answers/{id}
